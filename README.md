How to run WebdriverIO  <br>
a. Install nodejs <br>
b. Git clone <br>
c. Go to directory of project <br>
d. npm install <br>
e. npx wdio run ./wdio.conf.js <br>

This project uses the cucumber framework approach based on the .feature file in the feature folder <br>
